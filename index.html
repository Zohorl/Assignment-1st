The scope of a web developer is different than that of a web designer: A web designer's work focuses on improving the visual appearance of a website, and creating an intuitive user interface. Web designers build the structure, stylesheets, and layout on which the server-side implementation will be based. That said, many web developers are also capable or utilizing web design software such as Adobe Photoshop, Macromedia Dreamweaver, or Microsoft FrontPage to build their HTML websites.
